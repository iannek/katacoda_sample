## 対話式処理を行うシェルスクリプト  
シェルスクリプトファイル"test.sh"をエディタから作成し、下記の内容を書き込む。  

```
#!/bin/bash
echo "1文字以上入力してください"
read input

if [ -z $input  ] ; then
  echo "文字を入力してください"
else
  echo "適切な文字数です"
fi
```

test.shを実行する。  

`bash test.sh`{{execute}}

実行後、入力を求められるので任意の文字を入力してエンターを押す。  
下記のように表示されることを確認。(以下では"a"を入力している)  

```
$ bash test.sh
1文字以上入力してください
a
適切な文字数です
```

実行後、入力を求められるので何も入力せずエンターを押す。  
下記のように表示されることを確認。  

```
$ bash test.sh
1文字以上入力してください

文字を入力してください
```
